# comfy_face_clothes

# 목표
얼굴사진과 옷사진을 업로드하고, 옷의 유형, 체형에 대해서 입력을 받는다.  
업로드된 사진과 옷의 유형, 체형을 기반으로 이미지를 생성한다.

# 사전준비
ComfyUI 설치

# 실행방법
```
pip install -r requirements.txt
python run.py
```
http://127.0.0.1:8000
